# Typie
  Basic Typing Test Program. The logic can be used to make GUI games.

# Usage - Unix/Mac
  1) git clone https://github.com/shonber/Typie
  2) cd Typie/
  3) chmod 755 main.py
  4) python3 main.py
  5) Enjoy ;)

# Usage - Windows
  1) git clone https://github.com/shonber/Typie
  2) cd Typie/
  3) python main.py
  5) Enjoy ;)
